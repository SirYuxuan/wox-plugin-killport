# wox-plugin-killport

#### 介绍
本项目是一款Wox的插件
有关Wox请移步 [Wox](https://gitee.com/OrganizationStudy/Wox)

#### 安装教程

alt + space 呼出wox界面

输入

```
wpm install killport
```
![info](https://images.gitee.com/uploads/images/2019/1119/163815_04c03591_461804.png "20191119163756.png")

输入 kill port 进行使用

如 kill 8080
